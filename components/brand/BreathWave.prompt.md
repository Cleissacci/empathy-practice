The signature breath-wave motif — a teal + amber double sine wave. Lumina's most recognizable brand element.

```jsx
<BreathWave height={24} opacity={0.25} />   {/* subtle section divider */}
<BreathWave animated />                       {/* breathing loader */}
```

Use for dividers, loading states, progress accents, and illustration backdrops. Lower the `opacity` (~0.25) when used as a divider so it stays quiet.
