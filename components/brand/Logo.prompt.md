The Lumina logo — circular breath-wave mark plus wordmark. Use `reversed` on dark/teal surfaces.

```jsx
<Logo />                              {/* mark + wordmark */}
<Logo variant="reversed" size={40} /> {/* on teal header */}
<Logo showWordmark={false} size={48} />{/* icon only */}
```

Minimum mark size is 24px on screen. Keep ½× the mark height of clearspace around it.
