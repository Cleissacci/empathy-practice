Pill-shaped badge for module status, content tags, and learner levels. Pair `dot` with status; drop it for plain category tags.

```jsx
<Badge tone="teal" dot>In Progress</Badge>
<Badge tone="success" dot>Completed</Badge>
<Badge tone="amber">Emotional Intelligence</Badge>
<Badge outline>Video</Badge>
```

Tones: `teal` (active/info), `amber` (pending/intermediate), `rose` (needs attention/advanced), `success` (completed/beginner), `neutral` (not started). Never use color alone to convey status — the label carries the meaning.
