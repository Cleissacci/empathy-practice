A labelled metric card — the big number renders in Cormorant Garamond.

```jsx
<StatCard label="Reflection Streak" value={12} change="days in a row" />
<StatCard label="Empathy Score" value={82} tone="accent" change="↑ 4 pts this month" />
<StatCard label="Time Learning" value="4.2" suffix="h" change="↑ 38% vs last week" />
```
